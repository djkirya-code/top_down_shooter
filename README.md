Top-Down Shooter

 Описание
2D игра в жанре Top-Down Shooter, разработанная на движке Unity с использованием языка C#.  
Игрок управляет персонажем, перемещается по карте и уничтожает врагов с помощью стрельбы.

Проект выполнен в рамках курсовой работы.

  Технологии
- Unity
- C#
- Visual Studio

  Реализовано
- Управление персонажем (WASD)
- Прицеливание мышью
- Стрельба
- Логика врагов
- Система столкновений и урона

  Запуск
Открыть проект через Unity Hub  
или запустить сборку из папки `Build`.
